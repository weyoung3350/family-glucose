## 这次调研的方法与判断依据

- 采用的是“只读浏览 + 实际截图”的方式，不录入、不修改、不删除用户健康数据。
- 判断功能时，优先依据：
  - 实际页面标题
  - 页面内的按钮、字段、列表和标签
  - 页面之间的跳转关系
  - 图表/报表中的真实展示内容
- 对未继续深入的页面，只做“入口级确认”，不把推断写成已验证事实。

## 当前的关键判断

- “血糖记事本”不是单点记账工具，而是轻量慢病管理工具。
- 核心业务链路是：
  - `血糖记录 -> 趋势分析 -> 饮食/用药/运动联动 -> 家属协同 -> 导出与设置`
- 产品明显面向两类人：
  - 自我管理用户
  - 家庭照护/代管用户
- 关怀模式、亲友管理说明它有明显的家庭协同定位。
