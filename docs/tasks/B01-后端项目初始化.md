# B01 · 后端项目初始化

## 目标
搭建 FastAPI 后端项目骨架，能 `uvicorn app.main:app` 启动并通过 `/api/v1/health` 返回 200。

## 前置依赖
无。

## 上下文
- 概要设计 §3（技术栈）、§4.1（后端目录）、§10.1（依赖列表）、§11（配置）

## 输入
仅项目根目录（`./`）。本任务从零创建 `backend/`。

## 输出
新增以下文件：

```
backend/
├── pyproject.toml
├── .env.example
├── .gitignore
├── README.md
├── app/
│   ├── __init__.py
│   ├── main.py
│   ├── config.py
│   └── routers/
│       └── __init__.py
└── data/                 # 空目录占位（SQLite 文件将放此）
    └── .gitkeep
```

## 详细需求

### pyproject.toml
- `requires-python = ">=3.11"`
- 依赖锁定到概要设计 §10.1 列出的版本范围
- 包含 `[project.optional-dependencies] dev = ["pytest", "pytest-asyncio", "httpx"]`
- 命名 `glucose-api`，版本 `0.1.0`

### app/config.py
- 使用 `pydantic-settings` 的 `BaseSettings`
- 字段对齐概要设计 §11 .env 列表
- 类名 `Settings`，单例 `settings = Settings()`
- 默认值：`APP_ENV="development"`、`DATABASE_URL="sqlite:///./data/glucose.db"`、`LOG_LEVEL="INFO"`、`LLM_TIMEOUT_SEC=5`

### app/main.py
- 创建 `app = FastAPI(title=settings.APP_NAME, version="0.1.0", docs_url="/docs", openapi_url="/api/v1/openapi.json")`
- 配置 CORS：`allow_origins=settings.CORS_ORIGINS`（环境变量为逗号分隔字符串，需在 config 中转 list）
- 注册根路由 `GET /api/v1/health` 返回 `{"status":"ok","app":"家有糖人","version":"0.1.0","env": settings.APP_ENV}`
- 配置 logging（INFO 级别，格式同概要设计 §12）
- 异常处理器：FastAPI 的 `HTTPException` 转 `{ "code": exc.detail or "ERR_HTTP", "message": ... }`
- 请求中间件：注入 `request_id` 到 `request.state` 与响应头 `X-Request-Id`

### .env.example
完整复制概要设计 §11 的环境变量列表，敏感字段值留空。

### README.md
包含：
- 项目简介一行
- `cd backend && pip install -e .[dev]` 安装方法
- `cp .env.example .env` 然后编辑
- `uvicorn app.main:app --reload --port 8080` 启动
- `curl http://localhost:8080/api/v1/health` 验证

### .gitignore
忽略 `.env`、`data/*.db`、`__pycache__`、`.venv`、`*.egg-info`、`.pytest_cache`。

## 验收标准

- [ ] `cd backend && pip install -e ".[dev]"` 无错误
- [ ] `cp .env.example .env` 后 `uvicorn app.main:app --port 8080` 启动成功
- [ ] `curl localhost:8080/api/v1/health` 返回 `{"status":"ok",...}` 且 HTTP 200
- [ ] 响应头包含 `X-Request-Id`
- [ ] 浏览器访问 `localhost:8080/docs` 看到 OpenAPI UI（含 health 端点）
- [ ] 故意触发 `raise HTTPException(404, "ERR_TEST")` 后响应体为 `{"code":"ERR_TEST","message":"..."}`
- [ ] 项目根 `python -c "from app.config import settings; print(settings.APP_NAME)"` 输出"家有糖人"

## 不在范围内

- 任何业务路由（B03+）
- 单元测试模板（仅保证 dev 依赖可装）
