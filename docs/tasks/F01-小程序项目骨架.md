# F01 · 小程序项目骨架

## 目标
搭建微信原生小程序工程骨架，含 app.json、app.wxss、TabBar、登录入口与路由占位。所有 14 个页面文件已创建（即便内容是占位文字），TabBar 三个 tab 可点击切换。

## 前置依赖
无（可与 B01 并行）

## 上下文
- 概要设计 §4.2（前端目录结构）
- PRD §5（信息架构）

## 输入
- 概要设计中的页面清单（P01-P12）
- PRD 中的 TabBar 三 tab 名称

### app.js
```javascript
App({
  globalData: {
    token: null,
    user: null,
    family: null,
  },
  onLaunch() {
    const token = wx.getStorageSync("token");
    if (token) {
      this.globalData.token = token;
      this.globalData.user = wx.getStorageSync("user");
      this.globalData.family = wx.getStorageSync("family");
    }
    // 实际登录在 F02 + F03 任务实现
  }
});
```

### images/
TabBar 图标 6 张 PNG。要求：
- 尺寸 81×81 像素，PNG 透明底
- 风格统一：line icon，stroke 4px
- 默认色 #86909C，激活色 #FF9F40
- 图标语义：house（首页）、line-chart（分析）、user（我的）

可使用 https://lucide.dev 或 Tabler Icons 转 PNG。文件路径放 `miniprogram/images/`。

### 各 page 占位
每个 .wxml 仅一行 `<view class="placeholder">{页面名} 待实现</view>`，对应 wxss 一段简单居中样式。

## 验收标准

- [ ] 微信开发者工具打开本目录无编译错误
- [ ] TabBar 三 tab 可切换，激活态橙色
- [ ] 首页/分析/我的 三页可见对应占位文字
- [ ] 顶部导航栏显示"家有糖人"
- [ ] `getApp().globalData` 在控制台可访问
- [ ] images 目录下 6 张图标存在且不超过 30KB 每张

## 不在范围内
- 任何业务逻辑（F02+ 处理）
- 真实登录调用（F03 处理）
- AppID 填入（用户自行填）
