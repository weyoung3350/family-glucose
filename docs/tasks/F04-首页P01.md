# F04 · 首页 P01

## 目标
实现 P01 首页：欢迎语 + 今日已测次数 + 倒序记录流（按日期分组）+ 底部主操作按钮（"+ 记一次血糖" + 圆形麦克风快捷键）。

## 详细需求

### components/record-card
通用记录卡片，被首页和详情页（F05）复用。

属性：`record`（一个完整 RecordDTO）。

布局（参考原型）：
```
[ 8.2  mmol/L ]   [早餐后]                09:45
                  [👩 妈妈记 · 备注]      [一般]
```

- 大数字 `value`：颜色取自 `record.status.color`
- `mmol/L` 灰色小字
- 时段标签 chip（橙色浅底）
- 记录人头像（取昵称首字）+ "X 记" + 备注前 20 字
- 右侧：时间（HH:mm）+ 状态色胶囊（"理想/一般/偏高..."）
- 整张卡片 `bindtap` 抛出 `tap` 事件，父组件接收后跳转详情

### pages/index/index.js

- `onLoad` 与 `onShow` 都拉数据：
  - 调 `api.listRecords({ page:1, size:20 })`
  - 按 `measured_at` 的日期分组，组 label 用 `relative()`：今天 / 昨天 / N 天前 / YYYY-MM-DD
  - 计算 todayCount = 今天日期下的记录数
- 下拉刷新：page=1 重新拉
- 触底加载更多：page+1 拉到 hasMore=false
- `onAddTap` → `wx.navigateTo({ url:"/pages/add/add" })`
- `onMicTap` → `wx.navigateTo({ url:"/pages/ai-add/ai-add" })`
- `onCardTap` → `wx.navigateTo({ url:`/pages/detail/detail?id=${id}` })`

### images/mic.svg
SVG 图标，颜色 fill="currentColor"，与原型 P01 同款。或导出成 PNG（小程序支持 svg 但部分版本有兼容性问题，保险用 PNG 81×81）。

## 验收标准

- [ ] 进入首页能看到自己家所有记录，按时间倒序，分组到对应日期下
- [ ] 今天有 2 条记录时，右上"今日已测 2"
- [ ] 长按记录卡（首版可不做长按，仅 tap 跳详情）
- [ ] 底部 FAB 主按钮居中文字"+ 记一次血糖"，右侧麦克风圆按钮可见
- [ ] 点击麦克风 → 跳转 ai-add 页
- [ ] 点击主按钮 → 跳转 add 页
- [ ] 触底加载更多正常工作（mock 50 条）
- [ ] 空数据态（新家庭 0 条）显示提示
- [ ] 视觉与 P01 原型截图差异 ≤ 10%（颜色、间距、按钮尺寸）

## 不在范围内
- 详情页跳转后的删除/编辑能力（F05）
- 实时推送（首版只在 onShow 时刷新）
