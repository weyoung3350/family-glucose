## 详细需求

### P07 我的

页面结构：
1. 顶部用户卡（橙渐变）：左头像（昵称首字大圆 + 渐变底）+ 右昵称 + "X家 · 管理员/成员"
2. 菜单列表（白底圆角 cards 列表）：
   - 家庭管理 → `wx.navigateTo("/pages/family/family")`
   - 自定义血糖标准 → `wx.navigateTo("/pages/standards/standards")`
   - 数据导出 → `wx.navigateTo("/pages/export/export")`
   - 关于 → `wx.navigateTo("/pages/about/about")`

每条菜单：左侧圆形小图标（24×24，浅橙底圆 + 中文字 1 个，如"家""设""导""i"）+ 标签 + 右箭头 ›

每次 onShow 重新读 globalData，确保家庭状态变化后 UI 同步。

### P12 关于

页面结构：
1. 居中大 logo（80×80 圆角，"糖"白字渐变橙底）
2. 产品名"家有糖人"
3. 版本号"v0.1.0 · 家庭内部使用"
4. 菜单列表：
   - 使用说明 → 跳到一个内置 webview 或简单内容页（首版仅显示一段文字 modal）
   - 隐私说明 → 同上
   - 联系开发者 → 长按复制邮箱
5. 底部声明文字：
   ```
   本小程序仅供本家庭成员记录与查看血糖数据使用，
   不构成医疗建议，请遵医嘱。
   ```

版本号从全局常量 `APP_VERSION` 读取（utils/const.js 增加）。

## 验收标准
