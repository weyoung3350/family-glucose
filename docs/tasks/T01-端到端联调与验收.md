# T01 · 端到端联调与验收

## 目标
所有 B01-B10 与 F01-F10 任务通过单项验收后，做端到端集成验证，确认整个产品在真实环境下满足 PRD 第 12 节验收标准。

## 前置依赖
B01-B10 全部通过、F01-F10 全部通过、生产服务器已部署、AppID 已配置。

## 上下文
- PRD §12（验收标准）
- 概要设计 §7（认证流程）

## 输入
- 部署完成的后端（https://yourdomain/api/v1）
- 已上传到微信公众平台的小程序（开发版即可）
- 至少 3 个微信账号（模拟妈妈、女儿、儿子）

## 输出

```
docs/
└── tasks/
    └── _acceptance/
        ├── T01-端到端验收报告.md
        └── ... 各模块验收记录
```

## 详细测试用例

### TC01 · 三人加入家庭并协作记录
1. 妈妈打开小程序 → 创建"咱家"，拿到邀请码 → 复制
2. 女儿打开小程序 → 输入邀请码 → 加入成功，看到首页空数据态
3. 儿子打开小程序 → 输入邀请码 → 加入成功
4. 妈妈记录：8.2 / 早餐后 / 09:45 / 备注"二甲双胍" → 保存
5. 在女儿和儿子的小程序里下拉刷新 → 看到这条记录，且记录人显示"妈妈"

**通过条件**：5 秒内三端数据一致，记录人显示正确。

### TC02 · AI 一句话录入
1. 妈妈点击首页麦克风按钮进入 P02b
2. 文字输入"早上 9:20 空腹血糖 11.1，吃了二甲双胍"
3. 自动解析后 4 字段正确显示
4. 点击"确认保存"
5. 回到首页看到新记录，详情页有"AI 录入"标签

**通过条件**：解析 4 字段全对，记录入库带 source=ai。

### TC03 · 按天矩阵正确性
1. 准备数据：妈妈手动录入 30 天每天 2 条记录（混合状态）
2. 进入分析 → 按天矩阵 → 选最近 30 天
3. 抽查 5 个不同状态的格子，颜色与数值与原始数据一致
4. 抽查 3 个无记录格子显示 "-"

**通过条件**：颜色映射 100% 正确，数值显示正确。

### TC04 · 趋势图与报表
1. 选最近 30 天 / 全部时段
2. 折线图能渲染，点的颜色与状态对应
3. 测量次数与 30 天总记录数一致
4. 切到"报表" → 平均值与 chart.stats.avg 一致
5. 异常天数 = 人工统计结果

**通过条件**：所有数字 100% 准确。

### TC05 · 自定义标准变更
1. 妈妈进入 P10，把 fasting_high 从 6.1 改成 7.0 保存
2. 创建一条新记录 value=6.5 / fasting → 状态=理想
3. 女儿进入 P10：所有字段不可点，无保存按钮
4. 女儿创建一条 value=6.5 / fasting → 状态也=理想（同步生效）

**通过条件**：标准全家共享，成员视图正确受限。

### TC06 · 邀请码分享与移除
1. 妈妈在 P08 点"分享到微信"，发送给"邻居"账号（不让他真加入，只测试卡片）
2. 卡片显示"加入'咱家'，一起记录爸爸的血糖"
3. 妈妈在 P08 点击儿子的"移除"，二次确认通过
4. 儿子下次打开小程序进入 P09 加入家庭页

**通过条件**：分享卡片文案正确，移除即时生效。

### TC07 · CSV 导出
1. 进入 P11，选 30 天，点"生成并分享 CSV"
2. 通过微信传给电脑
3. Excel 打开，中文表头无乱码
4. 数据条数与小程序内 total 一致
5. 检查任意一行：日期、时间、时段、血糖值、状态、记录人、备注完全正确

**通过条件**：文件格式 100% 兼容 Excel，数据 100% 准确。

### TC08 · 异常路径
1. 用错误邀请码加入 → toast 提示
2. 创建已存在家庭名 → modal 内红字提示
3. 成员尝试调 PATCH /standards → 后端 403
4. 网络断开后操作 → toast "无法连接服务器"，应用不崩
5. token 过期后任意操作 → 自动重新 wx.login，请求重试一次成功

### TC09 · 5 分钟时间精度
1. 手动新增页时间 picker 选 09:23 → 不允许（步进 5 分钟，无 23 选项）
2. AI 录入"刚才血糖 7.8"（假设当前 09:23）→ 后端入库的 measured_at 为 09:25
3. 后端 PATCH /records 提交 measured_at=09:23 → 入库实际为 09:25

### TC10 · 视觉一致性抽检
对照 14 张原型截图：
- 每个页面截图 → 与 PRD 中嵌入的原型对比
- 颜色、间距、按钮尺寸偏差 ≤ 15%
- 中老年友好基线：正文字号 ≥ 32rpx，按钮高度 ≥ 88rpx
